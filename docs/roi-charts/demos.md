# ROI Comparison Chart
Chart the % ROI of one or many ICONOMI crypto funds or digital assets over a set period of time.

```html
<div id="roichart"></div>
```
```js
CryptoCharts.roiComparison({
  chart_id: "roichart",
  iconomi_tickers: ["BLX", "CAR"],
  cryptocompare_tickers: ["BTC", "ETH"],
  last_days: 14
});
```
